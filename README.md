# VT SpotMaker

ブラウザ上でスポットや連絡線を作成できる、VTオリシナ制作用のWebアプリです。

## 起動

このフォルダでローカルサーバーを起動します。

```sh
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 実装済み

- ワールド画像の読込
- `spot.png`、`route.png`、`route_esc.png`を標準表示素材として同梱
- スポット、通常連絡線、退却連絡線の表示画像を個別に差替え・リセット
- 通常・退却連絡線の太さを1〜16pxで調整
- 左クリックによるスポット追加、選択
- ドラッグ移動、右クリック削除
- Shift＋左クリックによる通常連絡
- Ctrl＋左クリックによる退却連絡
- ID、表示名、画像、継承、座標、サイズ、追加項目の編集
- 通常連絡、退却連絡の作成と削除
- 元に戻す・やり直す（画面ボタン、`Ctrl/Cmd + Z`、`Ctrl/Cmd + Shift + Z`）
- シナリオDATとスポットDATの直接読込
- DAT読込時の形式検査と、保存前のID・座標・接続先検査
- 読み込んだファイル名を維持したShift-JIS・CRLF形式のDAT保存
- 未保存状態の表示と、未保存のままページを閉じる際の警告
- IndexedDBへの自動保存と、起動時の復元・破棄確認
- ワールド画像と差し替え表示素材をBlob形式で自動保存
- PWAインストール、自動更新、オフライン起動
- シナリオDATとスポットDATの内容確認、個別書き出し
- シナリオDATは元の全文を保持し、直下の`spot`、`showSpot`、`linkSpot`、`linkEscape`だけを更新

元ツールと同じ `showSpot`、`linkSpot`、`linkEscape` 形式で書き出します。既存シナリオへ組み込む前にヘッダーの「内容を確認」で確認できます。

## PWA

GitHub PagesなどのHTTPS環境で公開すると、対応ブラウザではヘッダーの「インストール」からアプリとして追加できます。Web版とPWA版は同じURL・同じUIです。

一度オンラインで開くと、次回からオフラインでも起動できます。新しいバージョンは作業中に強制反映せず、アプリを閉じて次に開いた時に自動適用されます。
